## Overview

This project models an e-commerce platform's category structure as a **tree**.  
Each category is a node, and each subcategory is stored as a child node.

Example:

```text
All Products
├── Electronics
│   ├── Laptops
│   │   ├── Gaming Laptops
│   │   └── Business Laptops
│   └── Mobiles
│       ├── Android Phones
│       └── iPhones
└── Fashion
    ├── Men
    │   ├── Shirts
    │   └── Shoes
    └── Women
        ├── Dresses
        └── Handbags
